# Icon Setup Instructions

Since we cannot generate PNG files directly, you have two options:

## Option 1: Use Placeholder Icons (Quick Start)
The extension will work without custom icons. Chrome will use default icons.

## Option 2: Create Custom Icons (Recommended)

### Using Online Tool:
1. Go to https://www.favicon-generator.org/ or https://realfavicongenerator.net/
2. Upload any image or create a simple design
3. Generate icons in sizes: 16x16, 48x48, 128x128
4. Download and save them as:
   - `icons/icon16.png`
   - `icons/icon48.png`
   - `icons/icon128.png`

### Using Image Editor:
1. Open any image editor (Paint, GIMP, Photoshop, etc.)
2. Create three images with these sizes: 16x16, 48x48, 128x128
3. Use a dark background (#1a1d26) with a chart/graph symbol
4. Save as PNG files in the `icons` folder

### Design Suggestion:
- Background: Dark (#1a1d26)
- Symbol: Line chart or calculator icon in blue (#4a9eff)
- Keep it simple and recognizable at small sizes

## Option 3: Use the HTML Generator
1. Open `create-icons.html` in Chrome
2. It will automatically download the three icon files
3. Move them to the `icons` folder
4. Rename them to: icon16.png, icon48.png, icon128.png

For now, the extension will load with default Chrome icons and work perfectly fine!
